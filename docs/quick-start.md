# 快速开始

## 1. 打开购买页

- `https://opensale.chunlin.lat/`

选择套餐，填写邮箱，并按页面引导完成支付。

## 2. 支付成功后继续配对

支付成功后，继续进入配对选择页。

当前支持：
- Telegram
- Discord

## 3. 准备 Bot Token

### Telegram

- 在 Telegram 中搜索 `@BotFather`
- 使用 `/newbot` 创建机器人
- 获取 Bot Token

### Discord

- 在 Discord Developer Portal 创建应用
- 添加 Bot
- 获取 Bot Token
- 给 Bot 授予必要权限

## 4. 提交 Token

在对应配对页填写你的 Token 后，系统会自动把当前订单与渠道配置绑定起来。

完成后，你就可以开始在 Telegram 或 Discord 中使用 OpenClaw。
