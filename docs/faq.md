# 常见问题

## 1. 这个项目是什么

这是一个公开展示用仓库，主要用于说明 OpenClaw 云端版的使用方式、适用场景和购买入口。

## 2. 为什么有人会考虑云端版

常见原因包括：

- 想用 Claude Sonnet 4.5，但担心 Token 成本
- 不想自己准备服务器
- 不想自己准备域名
- 不想自己处理部署、网关和接入流程

## 3. 需要自己部署吗

常规购买与使用流程下，不需要自己从零部署整套环境。
